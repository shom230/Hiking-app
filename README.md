# 등산도감

전국의 산 뱃지를 수집하는 등산 기록 앱입니다.
GPS 인증을 통해 실제 방문한 산만 뱃지가 해금되며,
지역별로 산을 정복하고 나만의 도감을 완성해보세요.

---

## 주요 기능

- 대한민국 9개 지역 지도 기반 산 도감
- GPS 인증으로 실제 등산한 산만 뱃지 획득
- 지역별 산 뱃지 수집 및 도감 완성
- 산 정복 시 인증 사진 업로드
- 지역 완정복 시 특별 애니메이션 연출
- 뱃지 획득 수에 따른 칭호 시스템
- 테마 챌린지 시스템
- 누적 높이 및 획득 뱃지 통계
- 프로필에서 칭호 장착 및 관리

화면 비율: PC 모니터에서는 화면이 가로로 퍼져 보일 수 있으므로, 브라우저 창을 스마트폰 크기처럼 세로로 줄이거나 모바일 브라우저로 접속하시는 것을 권장합니다.

스와이프 제스처 기능 제한: 스마트폰 터치 기반의 '옆으로 스와이프' 제스처는 PC 마우스 드래그로 작동하지 않을 수 있습니다.

실시간 GPS 인증: 데스크톱 환경에서는 기기 특성상 GPS 위치 고도가 정확하지 않거나 위치 권한 허용이 필요할 수 있습니다. (※ 원활한 테스트를 위해 현재 테스트 모드가 활성화되어 있습니다.)

### 3. Expo Go 앱 설치
스마트폰에 Expo Go 앱을 설치해주세요.
- 아이폰: App Store에서 "Expo Go" 검색
- 안드로이드: Google Play에서 "Expo Go" 검색

##  실행 방법

### 1단계 — 저장소 다운로드
아래 링크에서 ZIP 파일로 다운로드하거나
터미널에서 아래 명령어를 입력해주세요.

git clone https://github.com/shom230/Hiking-app.git

### 2단계 — VS Code로 열기
1. VS Code 실행
2. 상단 메뉴에서 **File → Open Folder** 클릭
3. 다운로드한 **Hiking-app** 폴더 선택
4. **열기** 클릭

### 3단계 — VS Code 터미널 열기
VS Code 상단 메뉴에서
**Terminal → New Terminal** 클릭
(단축키: Ctrl + ` )

### 4단계 — 패키지 설치
VS Code 터미널에 아래 명령어 입력

npm install

설치가 완료될 때까지 기다려주세요. (1~2분 소요)

### 5단계 — 앱 실행

npx expo start

### 6단계 — 스마트폰에서 실행
터미널에 QR코드가 표시되면
아이폰 기본 카메라 앱으로 QR코드를 스캔해주세요.
Expo Go 앱이 자동으로 실행됩니다.

> ⚠️ 노트북과 스마트폰이 반드시
> 같은 와이파이에 연결되어 있어야 합니다.

## 사용법

1. 앱 실행 시 **대한민국 지도**가 표시됩니다
2. 지도를 **터치**하거나 **옆으로 스와이프**하면
   지역 카드 목록이 나타납니다
3. 원하는 **지역 카드**를 선택합니다
4. 해당 지역의 **산 뱃지 목록**이 표시됩니다
   - 회색 자물쇠: 아직 방문하지 않은 산
   - 컬러 뱃지: 이미 정복한 산
5. 뱃지를 클릭하면 **산 상세 정보**를 확인합니다
6. 실제 산 근처 **50m 이내**에서
   등산 완료 인증 버튼을 누르면 뱃지가 해금됩니다
7. 인증 시 **등산 사진**을 업로드할 수 있습니다
8. 지역의 모든 산을 정복하면
   **지역 완정복 애니메이션**이 나타납니다
9. 하단 **챌린지 탭**에서 테마 챌린지를 확인하세요
10. 우측 상단 **프로필 버튼**에서
    칭호를 확인하고 장착할 수 있습니다
